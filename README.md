# Svelte iTunes Artist Song Search
- Anonymous users can read/search 50 artist's songs 
- Anonymous users can read detailed song and play song preview

## Technologies
- Svelte/Sveltekit
- HTML
- CSS
- Tailwind CSS
- Javascript
- Typescript
